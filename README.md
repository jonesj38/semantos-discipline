# Semantos Discipline

Durable git-backed discipline instance for the Semantos codebase.

Source repository: <https://github.com/semantos/semantos-core>

- `sources/` — markdown source snapshots generated by `shad sources` for QMD indexing
- `artifacts/` — distilled Shad RLM outputs
- `runs/` — run logs and exported outputs
- `state/` — workflow/run state
